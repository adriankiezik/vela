# Vela

A clean-room **Minecraft Java Edition server** reimplementation in Rust.
Target: **protocol 776 (MC 26.2)**.

## Run

```sh
cargo run                      # binds 0.0.0.0:25565
cargo run -- 127.0.0.1:25599   # custom bind
```
